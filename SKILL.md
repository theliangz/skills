# 大模型视界微信公众号写作指南

## 概述

本 skill 专为公众号"大模型视界"（作者：视界君）设计，用于创作符合其独特风格的技术文章。

**公众号定位**：追踪 AI 前沿趋势，专注大模型与 Agent 技术演进，结合真实工程实践与论文解读，帮助技术人把前沿能力转化为可落地方案。

**核心能力**：除风格化写作外，本 skill 可自动从以下平台获取创作素材：

| 平台 | 用途 | 支持系列 |
|------|------|----------|
| AlphaXiv (alphaxiv.org) | 获取 LLM/Agent/深度学习高热度论文 | 优秀论文分享 |
| AIHot (aihot.today) | 获取当日 AI 领域热门新闻 | AI快讯分享 |
| GitHub (github.com) | 获取最热门 AI 相关开源项目 | 开源项目实战 |

## 文章风格规范

### 标题风格

采用双段式或三段式标题，突出核心技术亮点：

```
【模式1】主标题：副标题
示例：Kimi K2.5 重磅发布：原生多模态与 Agent Swarm 的完美融合，开源逼近 GPT-5.2 水平

【模式2】系列名称 | 主标题——副标题
示例：智能体架构实战（九）：思维树（ToT）规划——让 Agent 学会"深思熟虑"

【模式3】深度实践 | 主标题：副标题
示例：深度实践 | 构建高性能可执行 NL2SQL 系统：LLM + Milvus 向量检索工程全解
```

### 文章结构

```
大家好，我是视界君。

【开篇引入】1-2句话点明主题，激发阅读兴趣

---

### 01 / 一、章节标题
【正文内容】专业术语通俗解释，配代码/图表

### 02 / 二、章节标题
【正文内容】数据支撑，对比分析

---

## 总结
【核心观点提炼】

---

## 参考资料
- 论文链接/项目地址等

## 相关资源
- 系列名称标签

_喜欢本文？欢迎点赞、在看、转发，关注【大模型视界】，获取最前沿的AI论文硬核解析。_
```

### 语言风格规范

| 规范项 | 要求 |
|--------|------|
| **关键词强调** | 使用 `__关键词__` 双下划线加粗 |
| **专业性** | 技术术语准确，必要时中英对照 |
| **可读性** | 复杂概念用比喻/类比解释 |
| **数据支撑** | 性能数据、测试结果必须引用 |
| **对比分析** | 与竞品/基线对比说明优势 |
| **实战导向** | 代码示例、架构图、部署建议 |

### 格式细节

- **章节编号**：使用 `01`、`02` 或 `一、`、`二、` 格式
- **代码块**：使用三个反引号包裹，指定语言
- **图片说明**：使用 `_图片说明文字_` 斜体格式
- **分隔线**：主要章节间使用 `---` 分隔

## 四大文章系列

### 1. 智能开发实战系列

**特点**：深入 Agent 架构设计，提供完整代码实现

**结构模板**：
```

### 01 核心概念
### 02 实战实现（含代码）
### 03 深度分析
### 04 总结与权衡

### 03 落地影响

```

## 自动获取创作素材

当用户要求"今日创作"、"每日选题"或类似请求时，按以下流程自动获取素材：

### 工作流程

1. **识别用户需求**：确定需要创作哪个系列的文章
2. **获取素材**：从对应平台抓取热点内容
3. **筛选选题**：选择最具价值的内容
4. **创作文章**：使用对应模板完成写作

### 各平台获取策略

#### AlphaXiv 论文获取

**目标**：获取 LLM、Agent、深度学习相关的高热度论文

**操作步骤**：
1. 使用 webReader 访问 https://www.alphaxiv.org/
2. 筛选与 LLM/Agent/深度学习 相关的论文
3. 按热度排序，选择 Top 3-5 篇
4. 解析论文标题、作者、摘要、核心贡献
5. 推荐最具解读价值的 1 篇作为选题

**关键词筛选**：Large Language Model, LLM, Agent, Multi-Agent, Transformer, Attention, Fine-tuning, RLHF, Reasoning, Chain-of-Thought, RAG, Embedding, Diffusion, 等

#### AIHot 新闻获取

**目标**：获取当日最热门、最有价值的 AI 相关新闻

**操作步骤**：
1. 使用 webReader 访问 https://aihot.today/
2. 获取今日热门 AI 新闻列表
3. 筛选与大模型、Agent、AI 工程化相关的新闻
4. 分析新闻价值和受众关注度
5. 选择 1-2 条最具新闻价值的选题

**优先级排序**：产品发布 > 论文突破 > 开源项目 > 行业动态

#### GitHub 项目获取

**目标**：获取最热门、与 AI 高度相关的开源项目

**操作步骤**：
1. 使用 webReader 访问 https://github.com/trending
2. 筛选 AI/ML 相关语言（Python, Jupyter Notebook）
3. 选择 stars 数量增长最快、与 AI 高度相关的项目
4. 分析项目文档、核心功能、技术栈
5. 选择 1 个最具实战价值的项目

**筛选标准**：stars > 1000，与 LLM/Agent/RAG/推理/训练 相关，有清晰文档
